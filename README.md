# Gossip Forum

### File structure:
```
src/
├── components/
│   ├── auth/
│   │   ├── Login.tsx
│   │   ├── Register.tsx
│   ├── forum/
│   │   ├── Thread.tsx
│   │   ├── ThreadList.tsx
│   │   ├── Comment.tsx
│   │   ├── CommentList.tsx
│   ├── Navbar.tsx
├── services/
│   ├── api.ts
├── App.tsx
├── app.css
├── main.tsx
├── index.css
├── vite-env.d.ts
```
